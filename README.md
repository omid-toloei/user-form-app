# 👤 User Data Manager

A simple project for managing user information, built with pure **HTML**, **CSS**, and **JavaScript**.

## Features

- Add new users (name, last name, age, phone number)
- Data persistence with `localStorage`
- Delete all data with one click
- RTL layout with Persian font support

## Getting Started

No installation needed — just open `index.html` in your browser.
